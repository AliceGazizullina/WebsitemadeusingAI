# Сайт олимпиады (Python и Blender)

Сайт для проведения олимпиады по **Python для начинающих** и **Blender** с ограничением по времени и одной попыткой на трек. Результаты сохраняются в базу данных.

## Возможности

- **Два трека**: Python (для начинающих) и Blender
- **Ограничение по времени**: Python — 45 минут, Blender — 60 минут (настраивается в `config.py`)
- **Одна попытка**: каждый участник (по email) может пройти каждый трек только один раз
- **База данных**: SQLite по умолчанию; сохраняются участники, попытки, баллы и время

## Запуск

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Запустите сервер:
   ```
   python app.py
   ```

3. Откройте в браузере: http://127.0.0.1:5000

## Страницы

- **/** — главная (ссылки на олимпиаду и результаты)
- **/olympiad** — начало прохождения: ввод имени и email, выбор трека, вопросы, таймер, отправка ответов
- **/results** — таблица сохранённых результатов из БД

## База данных

Файл `olympiad.db` (SQLite) создаётся в корне проекта. Таблицы:

- **participants** — участники (email, имя)
- **attempts** — попытки (трек, баллы, время, ответы в JSON)

Для продакшена можно задать `DATABASE_URL` и `SECRET_KEY` в переменных окружения.

## Настройка вопросов и времени

- Вопросы и правильные ответы задаются в `app.py` в словаре `QUESTIONS` (ключи `python` и `blender`).
- Лимиты времени — в `config.py`: `PYTHON_TIME_LIMIT`, `BLENDER_TIME_LIMIT` (в секундах).
